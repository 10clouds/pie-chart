Yummy pies and donuts for AngularJS

What's that ?

Built on top of D3.js and AngularJS, n3-charts.pie-chart directive creates beautiful pie charts, donut charts and gauges in AngularJS applications. The result is simple, interactive and beautiful.

How to use ?

The GitHub page contains an exhaustive, comprehensive guide to use the pie-chart directive. For several use cases, please check the examples section.
